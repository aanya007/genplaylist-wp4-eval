# Project instructions for Claude Code

This repo tracks work on GenPlaylist WP-D Part 4 (evaluation matrix + 
prompt/lyric verbalization pipeline, adapted from VibeMus).

Rules:
- Always update PROGRESS.md after completing any meaningful step: check off
  the relevant box, and add a dated entry under ## Log describing what was
  done, what worked, what didn't, and what's next.
- Reference code lives in reference/GenPlaylist and reference/VibeMus — 
  treat these as read-only; copy/adapt code into src/ instead of editing
  in place.
- After updating PROGRESS.md, stage and commit changes with a clear message
  (don't push automatically — ask me first).
- Keep mock data in data/mock/, pipeline code in src/, generated outputs in outputs/.
